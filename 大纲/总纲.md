# 总纲

## 故事一句话
远古大贤者转世少女在科技消耗魔法的世界中，找回记忆的同时发现两千年的谎言，在真相与救赎之间寻找第三条路。

## 创意约束
- 反套路规则：AT-009 —— 主角的正确选择导致负面后果。星兰渴望找回记忆，但每段记忆都揭示前世罪责。记忆是诅咒，而非祝福。
- 硬约束（世界/能力/行为）：
  1. 记忆不可选择性解锁：触发不由星兰控制，每次被动接收完整片段
  2. 星火会的复国叙事与星兰的记忆真相存在结构性矛盾——真相公开将导致星火会信仰崩塌
  3. 大反派与星澜岛覆灭有直接渊源，但不是简单的"他是凶手"
- 主角缺陷：嫉恶如仇 + 不懂变通，经历灰色事件后慢慢接受复杂性
- 反派镜像：共享特质"都知晓历史真相" → 星兰选择面对与和解，反派选择彻底消灭魔法

## 核心主线
- 主线目标：星兰找回记忆 → 发现两千年谎言 → 面对星澜岛覆灭真相 → 阻止大反派借"秩序"之名实施极端计划 → 寻找魔法与科技共存的第三条路
- 主要阻力：银月帝国最高领导人的"社会达尔文"计划；黑暗势力的暗中操控；星火会复国主义的极端化倾向；魔法结晶石的不可再生性

## 核心暗线
- 暗线主题：星澜岛覆灭的真相——不是单纯的被侵略，而是王室禁忌魔法走向失控，大陆各国与星澜岛内部反对派共同参与的一场"终结"
- 回收节点：
  - 第1-15章：第一块记忆碎片，暗示星澜岛王室有秘密
  - 第30-40章：星火会登场，展示"官方叙事"
  - 第55-70章：第二块记忆碎片，揭示禁忌魔法的存在
  - 第85-100章：完整真相揭露，星兰面对前世罪责
  - 第110-120章：最终回收——真相如何改变各方势力的立场

## 反派分层（概要）
- 小反派（前期）：各处精英魔物、学院对手、低阶魔物暴乱
- 中反派（中期）：黑暗势力（可能与魔法结晶石黑市有关）、星火会内部激进派
- 大反派（后期）：银月帝国最高领导人——知晓魔法结晶石终将耗尽，推行"社会达尔文"以筛选"值得存活的人"

## 世界观/力量体系简介
- 世界观要点：2000年前星澜岛覆灭 → 魔法素散落世界各地 → 凝结为魔法结晶石 → 科技文明以魔法结晶石为能源崛起 → 魔法使用者几乎绝迹，仅少数天赋者保留微弱治疗能力 → 魔法结晶石不可再生，世界处于无声的倒计时
- 力量体系要点：
  - 远古魔法境界链：学徒→法师→大法师→贤者→大贤者（星兰前世为大贤者顶峰）
  - 现代科技装备体系：以魔法结晶石驱动，军衔制分级（与研究院职称并行）
  - 星兰金手指：星澜皇家血脉，前世大贤者记忆与天赋逐步觉醒，前期古典魔法→后期天赋直觉型

## 复合题材融合（可选）
- 主题材 / 副题材：魔法西幻（主） / 幻想言情（副）
- 融合机制（一句话）：主线剧情驱动世界观冲突与真相揭示，感情线嵌入主线成为情感锚点
- 节奏占比（建议 7:3）：剧情/世界观推进约70%，感情线约30%
- 关键冲突与爽点：
  - 冲突：科技vs魔法、真相vs信仰、星兰vs反派镜像
  - 爽点：记忆碎片揭示时的反转、身份揭露（迪化误解）、星兰与两位男主的情感推进、最终真相带来的世界观震撼

## 主角成长线
- 起点状态：失忆少女，在魔物暴乱遗址被发现，只记得名字，有微弱魔法天赋
- 关键跃迁节点：
  1. 第一次记忆碎片解锁（卷1末）→ 确认前世大贤者身份
  2. 古典魔法→天赋直觉的突破（卷2中）
  3. 禁忌魔法真相揭露（卷3末）→ 人格层面的完整
  4. 第三条路的创造（卷4末）→ 从"转世者"成为"开创者"
- 终局定位：不再只是星澜岛大贤者的转世，而是开创魔法与科技共存新纪元的奠基人
- （多主角）分工与冲突：姚宇代表科技理性，齐心代表守护意志，两人在星兰抉择时给出不同方向但都尊重她的选择

## 关键爽点里程碑
- 第3章：星兰首次无意识使用治愈术，周围人震惊
- 第8章：第一次与精英魔物战斗，天赋初显
- 第15章：第一个记忆碎片爆发，确认大贤者身份
- 第25章：与姚宇首次正面交锋（研究院vs魔法觉醒者）
- 第35章：星火会登场，揭示"复国"使命
- 第50章：记忆碎片揭示星澜岛王室的秘密
- 第70章：禁忌魔法真相片段，星兰道德崩溃与重建
- 第90章：大反派完整计划揭露，反派镜像对决
- 第110章：星兰做出第三条路的选择
- 第120章：终局——和解而非毁灭
